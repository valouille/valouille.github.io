Installer HAproxy sous Debian Wheezy

2013-06-05
linux sysadmin

Contents

HAproxy n’est plus dans les depots Debian Wheezy, car l’équipe n’avait pas corrigé certains bugs.

Il est cependant possible de l’installer en ajoutant les backports. Pour cela, simplement ajouter la ligne suivante dans le fichier /etc/apt/sources.list :

deb http://ftp.debian.org/debian/ wheezy-backports main

Et :

apt-get update ; apt-get install haproxy
