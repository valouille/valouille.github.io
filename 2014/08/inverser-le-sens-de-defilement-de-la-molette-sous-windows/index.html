Inverser le sens de défilement de la molette sous Windows

2014-08-14
windows

Contents

Sous Mac OS X, par défaut, le sens de défilement est inversé. Cela s’appelle le « natural scrolling » ou « défilement naturel ». Quand on passe d’OS X à Windows régulièrement, on peut préférer avoir le même comportement sous Windows.

Afin d’activer cette option, il faut lancer la commande Powershell ci-dessous (à lancer dans l’application « Windows Powershell » en tant qu’Administrateur)

Get-ItemProperty HKLM:\SYSTEM\CurrentControlSet\Enum\HID\*\*\Device` Parameters FlipFlopWheel -EA 0 | ForEach-Object { Set-ItemProperty $_.PSPath FlipFlopWheel 1 }

Le redémarrage de l’ordinateur est nécessaire afin d’appliquer les modifications.

La deuxième solution est de le faire à la main. Il faut lancer l’éditeur de registre (Touche Windows + R, et entrer « regedit »)

Il faut éditer la clé suivante :

HKEY_LOCAL_MACHINE\SYSTEM\CurrentControlSet\Enum\HID\VID_***\VID_***\Device Parameters

Changer la valeur de FlipFlopWheel de 0 à 1 et redémarrer l’ordinateur.

Afin de trouver le VID, il faut aller voir dans le « Panneau de configuration » (Afficher par : Grandes icones), et ouvrir l’outil « Souris ». Ensuite, dans l’onglet « Matériel », cliquer sur « Propriétés ». Dans l’onglet « Détails », choisir la propriété « Chemin d’accès à l’instance du périphérique ».
