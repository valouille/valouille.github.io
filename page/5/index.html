Changer l’emplacement du répertoire Documents sous Windows

2014-08-14
windows
Les dossiers « Documents », « Musique », « Téléchargements » … sont originalement placés dans C:. Si on souhaite déplacer cet emplacement sur un autre disque ou un autre partition, tout en conservant ses documents, il faut procéder comme ci-dessous : Ouvrir l’explorateur Windows (Touche Windows + E) Faire un clique droit sur « Documents » et choisir « Propriétés » Aller dans l’onglet emplacement Changer l’emplacement et valider en cliquant sur « OK » Si Windows demande s’il faut créer le dossier, faire « Oui » Ensuite, Windows va demander s’il faut déplacer les fichiers dans le nouvel emplacement, cliquer sur « Oui » Il n’y a plus qu’à attendre la fin de la copie.
